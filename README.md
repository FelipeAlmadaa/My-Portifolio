# Felipe Almada | Portfolio

Welcome to the source code of my professional portfolio! I am a **Senior Software Engineer** and **Senior QA Automation Engineer** with over 5 years of experience building scalable backend solutions and robust automation frameworks.

## 🚀 About the Project

This is a personal portfolio website designed to showcase my projects, skills, education, and career goals. The interface is fully responsive, featuring multiple layout experiments, smooth animations, and a modern aesthetic with glassmorphism effects.

## 🛠️ Built With

- **HTML5 & CSS3**: Custom grid architectures (including Bento Box layouts), modern flexbox, and responsive media queries.
- **Vanilla JavaScript**: Lightweight DOM manipulation, custom cursor effects, and intersection observers for scroll animations.
- **DotLottie**: Integrated lottie animations for interactive elements.

## 📂 Project Structure

- `/CSS`: Contains all the modular stylesheets for the project (e.g., `style.css`, `home.css`, `projects.css`, `responsive.css`).
- `/JS`: Contains JavaScript logic (e.g., `skeleton.js` for loading effects).
- `/IMGS`: Assets, icons, and personal photos.
- `/FILES`: Downloadable documents such as my resume.
